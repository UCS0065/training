# training

今天要教大家如何操作Branch，這也是Git絕對要會的技能，總共分為以下幾個功能：

* 新增及切換Branch
* 合併Branch
* 與GitHub整合

## 新增及切換Branch

* `git branch NEWBRANCH`：開一個叫做`NEWBRANCH`的Branch
* `git checkout NEWBRANCH`：切換到一個叫做`NEWBRANCH`的Branch
* `git checkout -b NEWBRANCH`：開一個叫做`NEWBRANCH`的Branch，並且直接切換過去
